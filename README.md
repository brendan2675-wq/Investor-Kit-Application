# Investor Kit Application

A property buying process knowledge framework website built with HTML, CSS, and JavaScript.

## Files

- `Investorkit.html` - Main HTML file containing the investor kit website
- `profile picture.png` - Profile image
- `New Header Image 1.jpg` - Header background image
- `new brain.png` - Brain icon for the learnings section

## Setup

Simply open `Investorkit.html` in a web browser to view the website.
